# Data Science Salary Dashboard – Excel Project

This project is an interactive Excel dashboard that explores global data science job market trends. It focuses on salary distribution across various job titles, countries, and job types using intuitive Excel features like slicers, pivot tables, and charts.

## 📊 Dashboard Features

- Filter by **Job Title**, **Country**, and **Job Type**
- Compare salary ranges across multiple roles (e.g., Data Analyst, ML Engineer)
- See global job distribution on a map
- Key KPIs: 
  - Median salary
  - Top job platform
  - Total job count

## 📁 Files Included

- `1_Salary_Dashboard.xlsx` – The main Excel dashboard with slicers and visuals
- `data_jobs_salary_all.xlsx` – Raw data used to build the dashboard
- `dashboard.png` – A visual preview of the dashboard
- `README.md` – This documentation file

## 🛠️ Tools Used

- **Microsoft Excel**  
  - Pivot Tables  
  - Pivot Charts  
  - Slicers  
  - Conditional Formatting  
  - (Optionally Power Query, if used)

## 📌 Key Insights

- Median Salary for a Data Analyst in the U.S.: **$90,000**
- Most common job type: **Full-time**
- Top job platform: **Indeed**
